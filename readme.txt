Add a free simple customizable Quebec income tax calculator to your web site. 

== Description ==

This is simple [Quebec tax calculator](https://calculatorscanada.ca/quebec-income-tax-calculator/) calculates income taxes for residents of Quebec province in Canada for 2021 fiscal year. Latest tax rates can be found [here](https://calculatorscanada.ca/quebec-tax-brackets-rates/).

In case somebody needs here are plugin versions for earlier fiscal years:
* 2020 year - there is no plugin for 2020 Quebec income tax
* 2019 year ([v2019.1](https://plugins.svn.wordpress.org/cc-quebec-tax-calculator/tags/0.2019.1/))
* 2018 year ([v2018.1](https://plugins.svn.wordpress.org/cc-quebec-tax-calculator/tags/0.2018.1/))
* 2017 year ([v2017.1](https://plugins.svn.wordpress.org/cc-quebec-tax-calculator/tags/0.2017.1/))
* 2015 year ([v2015.1](https://plugins.svn.wordpress.org/cc-quebec-tax-calculator/tags/0.2015.1/))
* 2014 year ([v2014.2](https://plugins.svn.wordpress.org/cc-quebec-tax-calculator/tags/0.2014.1/))
* 2013 year ([v2013.2](https://plugins.svn.wordpress.org/cc-quebec-tax-calculator/tags/0.2013.1/))

Calculator is very easy customizable: you can change color of background, borders and text to match your web site's theme and change widget title.

== Installation ==

1. Extract the zip file and just drop the contents in the wp-content/plugins/ directory of your WordPress installation
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to Appearance -> Widgets and add the widget to your website sidebar

OR

Use [cc_income_tax_qc] short code if you want embed the mortgage calculator into a post instead of adding it as a widget on sidebar.

Short code parameters are:

  * title (optional) - set calculator's title (default - 'Ontario tax calculator')
  * bg_color (optional) - set background color (default - '#f8f8f8')
  * border_color (optional) - set border color (default - '#ddddd')
  * text_color (optional) - set text color (default - '#000000')
  * dev_credit (optional) - show developer's credit (default - '1') 

Example of shortcode usage: 

[cc_income_tax_qc title='Income tax calculator' bg_color='#eaf4d7' border_color='#94c738']

Use [Color Picker](http://www.colorpicker.com/) to get hex code of color you need. 

Please visit [plugin home page](https://calculatorscanada.ca/income-tax-calculator-wordpress-widget/) for more detailed installation and setup instructions
